# 📚 Library Management System

A RESTful Library Management System developed using Spring Boot. This project provides CRUD (Create, Read, Update, Delete) operations for managing books in a library. It is built using Spring Boot, Spring Data JPA, Hibernate, MySQL, and Maven.

---

## 🚀 Features

- Add a new book
- View all books
- View a book by ID
- Update book details
- Delete a book
- REST API implementation
- MySQL database integration
- Flyway database migration support

---

## 🛠️ Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Flyway
- REST API
- Postman

## 📡 API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /books | Get all books |
| GET | /books/{id} | Get book by ID |
| POST | /books | Add a new book |
| PUT | /books/{id} | Update book |
| DELETE | /books/{id} | Delete book |
